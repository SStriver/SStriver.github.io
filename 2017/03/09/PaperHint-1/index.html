撰写英文论文时需要注意的地方

1.eg.

to do => with

2.时态

描述我的工作时 => 过去式
描述结论时 => 现在式

描述他人工作时一般使用现在时或者过去完成时，但是作者作为主语通常使用过去式。

表格或者图作为主语时，通常使用现在时。描述计算和统计结果 => 现在时态。

3.去ing：

ing结尾一般表示名词。

4.冠词(the, a, an)：

1.特指双方都明白或者上文提及的人或者事。

2.独一无二

3.单数名词前面

4.最高级或者形容词前面

5.与复数名词连用指该类别

5.This+名词

6.eg.

Then the solution was placed. =>

The solution was then placed.

7.eg.

The new group of drugs were… =>

The new group of drugs was…

8.eg.

a modified cognitive-function test =>

a modified test of cognitive function

9.eg.

don’t => do not

can’t => cannot
